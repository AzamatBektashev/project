6. Data Visualization

Create a bar chart to visualize the survival rate by passenger class.

# Bar chart for survival rate by passenger class

plt.figure(figsize=(8, 6))

plt.bar(survival_rate_by_class.index, survival_rate_by_class.values, color=['gold', 'silver', 'brown'])

plt.xlabel('Passenger Class')

plt.ylabel('Survival Rate')

plt.title('Survival Rate by Passenger Class')

 

#provides the positions for the ticks, and the list ['First Class', 'Second Class', 'Third Class'] provides the labels.

plt.xticks(survival_rate_by_class.index, ['First Class', 'Second Class', 'Third Class'])

plt.ylim(0, 1)  # Set y-axis limit from 0 to 1

plt.grid(axis='y')  # Add gridlines

plt.show()
